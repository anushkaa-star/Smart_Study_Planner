# Smart Study Planner

## Overview
A Java console-based application to manage study tasks with deadlines and priorities.

## Features
- Add tasks
- View tasks
- Mark tasks as completed
- Delete tasks
- Priority system (High/Medium/Low)

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- Exception Handling

## How to Run
1. Compile:
   javac *.java
2. Run:
   java SmartStudyPlanner
